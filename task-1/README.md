# SWOCTS — Task 1

## The Story

You've been handed credentials to what looks like a routine sysadmin web panel. It's running on some internal box that nobody seems to have touched in a while. Poke around — there's more here than the login page suggests.

---

## Description

A Python Flask web application simulating a Linux server administration dashboard. Players work through a short chain of reconnaissance techniques to collect three flags.

- **Difficulty:** Easy
- **Category:** Web / OSINT
- **Flags:** 3

---

## Running Standalone (single instance)

### Prerequisites
- [Docker](https://docs.docker.com/get-docker/) with Compose plugin

### Start

```bash
cd task-1
docker compose up --build -d
```

Available at **http://localhost:5000**.

### Stop

```bash
docker compose down
```

---

## Running Multi-Team (with manager)

> **Recommended:** use `setup.sh` at the repo root — it pulls the pre-built image from GHCR and configures everything automatically. See the root `README.md`.

The manager passes `FLAG_SOURCE`, `FLAG_SSH_CREDS`, and `FLAG_BASH_HISTORY` as environment variables when spinning up each team's container. Each team receives unique per-team flag values derived from `FLAG_SECRET` + team name.

---

## Flags

| Env var | Location | Points |
|---------|----------|--------|
| `FLAG_SOURCE` | Embedded in the login page HTML source | 75 |
| `FLAG_SSH_CREDS` | Hidden in EXIF metadata of an image inside `files.zip` | 150 |
| `FLAG_BASH_HISTORY` | Visible in the simulated bash history on the dashboard | 200 |

**Customising flags for standalone testing:**

Edit `docker-compose.yml` before starting:

```yaml
environment:
  FLAG_SOURCE:       "CTF{your_flag}"
  FLAG_SSH_CREDS:    "CTF{your_flag}"
  FLAG_BASH_HISTORY: "CTF{your_flag}"
```

## Generating Artifacts

`files.zip` (served on the dashboard for download) is generated by `generate_artifacts.py`. It embeds `FLAG_SSH_CREDS` into image EXIF metadata. The GitHub Actions workflow runs this automatically before building the Docker image.

To regenerate locally:

```bash
pip install Pillow piexif
python generate_artifacts.py
```

`static/files.zip` is gitignored — it must be present before the Docker image is built.
